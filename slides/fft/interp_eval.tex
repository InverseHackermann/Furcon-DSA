\frame{

\begin{columns}
\column{0.5\textwidth}

\begin{itemize}
\item For a set of $n$ points, there is a unique polynomial with degree less than $n$ that passes through all the points
\item Evaluating the polynomial at $n$ different $x$ values to find these points typically takes $O(n^2)$ time
\item If we choose the $x$ values cleverly, we can do better
\end{itemize}
