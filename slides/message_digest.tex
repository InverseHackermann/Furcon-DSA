\begin{wideitemize}
\item Also called ``hash functions"
\item Takes any size input, makes fixed-size output
\item Ideally, different inputs make different outputs
\item Hard to find two inputs with same output for cryptographic hash functions
\item e.g. MD5, SHA, BLAKE
\end{wideitemize}
